<h3>How to run the Cpp Code?</h3>
<ol>
  <li>Install a C++ compiler such as MinGW for Windows or use g++ on Linux and Mac.</li>
  <li>Open the terminal and navigate to the folder containing your C++ file using the cd command.</li>
  <li>Compile the program by typing g++ filename.cpp -o output</li>
  <li>Run the program by typing
    <ul>
      <li>output.exe on Windows</li>
      <li>./output on Linux or Mac</li>
    </ul>
  </li>
</ol>
